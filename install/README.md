# ggroupid: aartifactid

aartifactid project

## 0. Requirements
- Docker
- docker-compose

## 1. Configuration
Modify `.env` file with your params if needed.

## 2. Run
To run the app standalone:

`make run`

To run the app and all required services:

`make up`

You can also run custom profile with:

`make run profile`

`make up profile`
